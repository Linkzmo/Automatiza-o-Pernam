Objetivo:
Um Código que consiga fazer uma análise de produtos na web e fazer um comparativo de preços e trazer a melhor opção



Bibliotecas:

Beautiful Soup 
-Beautiful Soup é uma biblioteca Python para extrair dados de arquivos HTML e XML. 
Ela funciona com seu analisador favorito para fornecer maneiras idiomáticas de navegar

Pandas
-Em programação de computadores, pandas é uma biblioteca de software criada para a linguagem Python para manipulação e análise de dados. 
Em particular, oferece estruturas e operações para manipular tabelas numéricas e séries temporais. 
É software livre sob a licensa licença BSD.[2] O nome é derivado do termo inglês "panel data"(dados em painel), 
um termo usado em estatística e econometria para conjunto de dados que incluem várias unidades amostrais

Selenium
-O Selenium é uma poderosa ferramenta de automação para navegadores web que permite realizar tarefas repetitivas de forma programática.
